# AI PDF Analyzer Web App

## 🚀 Overview
A Flask-based web application that allows users to upload PDF files and get AI-powered analysis using Gemini API.

---

## ✨ Features
- Upload PDF files
- Extract text from documents
- AI-based processing using Google Gemini API
- Simple and clean web interface

---

## 🛠️ Tech Stack
- Python
- Flask
- Gemini API
- HTML

---

## 📦 Installation

```bash
pip install -r requirements.txt
python app.py
